# fractest

A Javascript Fractal Generator. Features Include

* Formulas
  * Mandelbrot
  * Julia Set
  * Newton
  * IFS-like (Serpenski Triangle, Snowflake, Fern, etc)
  * Fibonacci Spirals
* Color Maps
* Zooming
* Anti-Aliasing
* High-quality Image Export

More information at https://www.fiforms.net/tech/museum/fractest
